# Knowledge Graphs

- Prefer `graphify-out/GRAPH_REPORT.md` over broad raw-file exploration when it exists.
- Prefer `graphify-out/wiki/index.md` when a wiki has been generated.
- Keep architecture docs cross-linked so both graphify and Obsidian Graph remain useful.
- If the project uses Obsidian, enable the core Graph view plugin and use Local Graph to inspect note neighborhoods before reading large note sets.
