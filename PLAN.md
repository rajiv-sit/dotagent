# PLAN

## Purpose

`PLAN.md` is the working execution tracker for the project.

Use it to capture:

- current objective
- completed work
- in-progress work
- next steps
- blockers
- verification status

## Recommended Structure

### Current Objective

One short statement of the active goal.

### In Progress

- item

### Next

- item

### Blockers

- item

### Verification

- tests run
- manual checks
- known gaps
